##  Installation

<img src="resources/nodejs.png" alt="nodejs" style="width: 40%;"/>

Yo and the generators used are installed using npm.

<div class="fragment">
    <p>Install Yeoman:</p>
    <pre><code data-trim data-noescape>
        npm install -g yo
    </code></pre>
</div>

<div class="fragment">
    <p>Install a Generator:</p>

    <pre><code data-trim data-noescape>
        npm install -g generator-webapp
    </code></pre>
</div>
