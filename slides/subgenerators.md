##  Sub-generators

Yeoman has Sub-generators to be used with the created scaffolding

<div class="fragment">
    <p>To add new Slides:</p>
    <pre>
        <code data-trim data-noescape>
            yo reveal:slide "Slide Title"
        </code>
    </pre>
</div>
<div class="fragment">
    <p>With markdown:</p>
    <pre>
        <code data-trim data-noescape>
            yo reveal:slide "Slide Title" --markdown
        </code>
    </pre>
</div>
